# WAO Hub

WAO Hub is an ultra-lightweight ephemeral proxy that seamlessly connects different types of servers and browsers.

Anyone can launch it both locally and remotely.

![](/images/wao-hub.png)

WAO Hub:

- Connects the browser to remote HyperBEAM nodes via WebSockets
- Syncs your local file system to the browser via WebSockets
- Relays local AO tests to the browser via WebSockets
- Connects browsers with other browsers via WebRTC to create P2P mesh networks

```bash
npx wao hub
```
